# Demo Apps

Demo apps are self-contained and complete applications that resembles real-world real-time analytics use cases. They have schema, table, and index definitions baked-in, along with an assortment of event simulators to simulate realistic traffic.

Currently, there is one demo app available:

* [All About That Dough Pizza Shop](demo-apps/pizza-shop.md)
