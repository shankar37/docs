## Query Data Source

<Tabs items={['Simple Website', 'Website with Anomalies']}>
  <Tabs.Tab>
   To have a look at the data that we've imported, click on the **Query Console** link, which will open the [Pinot Data Explorer](https://docs.pinot.apache.org/basics/components/exploring-pinot).
Click on the table name that you used (we use the name _anomalies_ in this guide) and then click **RUN QUERY** to run a basic query against the data source:

![Query simple Data Source](/img/sample-simple-query.png)
*Query simple Data Source*

Below are some other queries that you can run against this dataset:

```sql
select sum(views)
from simple
```
*Total views*

```sql
select count(*)
from simple 
WHERE hoursSinceEpoch > 770500
```
*Recent events*

  </Tabs.Tab>
  <Tabs.Tab>
      To have a look at the data that we've imported, click on the **Query Console** link, which will open the [Pinot Data Explorer](https://docs.pinot.apache.org/basics/components/exploring-pinot).
Click on the table name that you used (we use the name _anomalies_ in this guide) and then click **RUN QUERY** to run a basic query against the data source:

![Query anomalies Data Source](/img/sample-anomalies-query.png)
*Query anomalies Data Source*

Below are some other queries that you can run against this dataset:

```sql
select browser, sum(views) AS totalViews
from anomalies 
group by browser
order by totalViews DESC
```
*Top browsers*

```sql
select country, count(*) AS count
from anomalies 
where clicks > 0
group by country
order by count DESC
```
*Most popular country*
  </Tabs.Tab>
