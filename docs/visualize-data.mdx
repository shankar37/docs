# Visualize data

Visualize data hosted in your StarTree Cloud instance of Apache Pinot using one of the following methods:

- Use the [Superset](https://docs.pinot.apache.org/integrations/superset) integration
- Use the [Tableau](https://docs.pinot.apache.org/integrations/tableau) integration
- Use [Redash](https://redash.io/):
  - **To use Apache Pinot with Redash**, you must set up a Python data source in Redash, and write queries in Python to invoke the Pinot broker through the `pinotdb` Python client library. Follow the instructions to [visualize data with Redash](./visualize-data/visualize-data-with-redash).
- Visualize monitoring metrics with out-of-the-box [Grafana dashboards](./visualize-data/visualize-data-with-grafana.mdx).
